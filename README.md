## 프로젝트에서 다루는 주요 개념
* footer 하단 고정(컨텐츠 길이가 변할 때 마다 달라지는 footer)

1.footer {
    position : absolute;
    bottom : 0;
    }

2.footer {
    position : fixed; (상시고정)
    bottom : 0;
}
3.강제로 컨텐츠 길이를 늘이고, 그 아래에 footer를 박는다.
<transform : translateY(-100%);> 를 통해 footer의 높이만큼 footer를 강제로 올려치기하여 해결한다.
